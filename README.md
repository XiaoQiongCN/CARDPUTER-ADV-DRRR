# CARDPUTER-ADV-DRRR
DRRR.COM聊天室的CARDPUTER-ADV客户端

我想要创建一个DRRR.com聊天室的客户端，目前仅仅有这个想法，并学习相关知识。

计划：

1.考虑学习方向micopython或是c++，之前有python的少量经验。

2.实现wifi的搜索、连接、保存，手动重置。

3.实现drrr.com的访问，id设置，房间列表获取（设定时间刷新，或是手动刷新）。

4.进入房间、退出房间（避免长时间占用服务器）。

5.进入房间后，实时获取其他人员发送的聊天内容（中文文本支持）。

6.Card键盘输入支持，中文输入法应该较难实现。

7.drrr中其他人员发送的背景音乐播放。

8.sd卡中txt创建音乐连接列表，便于在drrr聊天房间中选择进行发送使用。

9.Card当前状态，电源电量显示、网络状态显示、背景亮度设置、消息提示音。

目前这个项目并没有进展，同时由于仅业余时间可以推进，可能进度缓慢（每月？？）。
